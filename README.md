# Weather App 🌦️

This is a simple weather app that shows current weather using OpenWeather API.

## 👨‍💻 Technologies Used:
- HTML
- CSS
- JavaScript

## 📦 Features:
- Search city weather
- Temperature, humidity, wind info

## 🚀 How to Run:
Just open `index.html` in your browser
